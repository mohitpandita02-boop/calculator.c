# C Calculator

A simple and beginner-friendly **calculator program written in C**, capable of performing the following operations:

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)

This project is created to practice basic C syntax, user input, and switch-case logic. Suitable for beginners learning C or uploading their first project to GitHub.
